```
以服务器方式运行dist文件下的index.html
```

## 查看/修改项目具体实现代码
```
npm install(下载相关依赖Project setup)
```
npm run serve(运行查看Compiles and hot-reloads for development)
```
npm run build(打包Compiles and minifies for production)
```
